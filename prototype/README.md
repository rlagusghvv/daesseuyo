# Prototype Notes

## Direction

현재 플레이 가능한 방향은 `대쓰요: Clutch Duel`, 웹 기반 1타석 결투 + 클러치 로그라이크다.

브라우저 Canvas MVP는 언리얼 리얼 3D 야구를 대체하는 현실적인 첫 플레이어블이다. 고품질 선수 애니메이션 대신 한 타석의 판단, 조준, 타이밍, 결과 납득성에 집중한다.

## Web MVP Goal

9회말 2아웃 같은 클러치 상황을 짧은 결투로 만들고, 5개 상황을 보상 카드와 함께 연속 돌파하게 만든다.

## Scope

- HTML Canvas 경기 화면
- AI 생성 경기장, 투수, 타자, 포수 컷아웃
- 커스텀 투구 궤적과 구종별 변화
- 존 승부/유인구/몸쪽/터널링 예측 심리전
- 조준/타이밍 기반 타격 판정
- 투구 시작과 스윙 직접 조작
- 카운트, 아웃, 주자, 점수
- 5개 클러치 타석 런
- 성공 후 보상 카드
- 듀얼 코드 기반 비동기 챌린지

## Validation

- 투구 의도와 공 궤적을 읽는 재미가 있는가?
- 예측 적중/실패가 즉시 이해되고 판정 차이로 느껴지는가?
- 타격 결과가 조준과 타이밍으로 납득되는가?
- 실패해도 다시 한 타석을 누르고 싶을 정도로 루프가 짧고 선명한가?
- 같은 듀얼 코드를 공유했을 때 같은 패턴의 도전으로 느껴지는가?
- 한국 야구의 야간 경기와 클러치 감정이 살아나는가?
