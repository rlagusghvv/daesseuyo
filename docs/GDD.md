# 대쓰요 : real BaseBall

## One-Line Pitch

실제 KBO IP 없이, 한국 프로야구의 감정과 경기 리듬을 리얼 3D 액션으로 구현하는 Unreal Engine 5 기반 야구 게임.

## High Concept

MLB에는 더 쇼가 있고, NPB에는 프로야구 스피리츠가 있다. 한국 야구에는 그 급의 리얼 직접 플레이 야구 게임이 없다.

이 프로젝트는 그 빈자리를 겨냥한다. 목표는 단순한 카드 수집이나 감독 시뮬레이션이 아니라, 투수의 릴리스, 타자의 스윙, 포수의 블로킹, 내야수의 첫 발, 외야수의 펜스 플레이까지 조작하고 납득할 수 있는 리얼 야구 액션이다.

## Working Title

**대쓰요 : real BaseBall**

의미:
- 한국 야구 중계 특유의 말맛과 끝내기 감정을 떠올리게 하는 제목
- "real BaseBall"로 더 쇼/스피리츠 계열의 리얼 직접 조작 야구 방향을 명확히 함
- 실제 해설자 이름, 음성, 초상, 방송 그래픽을 직접 사용하지 않는다.

국가대표 영광의 순간 모티브는 [Naming and Motifs](NAMING.md)에 모드/시나리오 후보로 정리한다.

## License Strategy

이 게임은 실제 KBO, MLB, NPB, 국가대표, 올림픽, WBC, Premier12, 방송사, 해설자 IP를 사용하지 않는다.

- 실제 구단명 사용 안 함
- 실제 구단 로고, 색상 조합, 유니폼 사용 안 함
- 실제 선수 이름, 얼굴, 기록 사용 안 함
- 실제 구장명과 고유 외형 사용 안 함
- 실제 대회명, 엠블럼, 방송 그래픽 사용 안 함
- 실제 응원가, 구단 구호, 대표팀 문구 사용 안 함
- 실제 해설자 이름, 음성, 초상, 고유 방송 자료 사용 안 함

대신 한국 야구의 구조와 감정은 가져온다.

- 10개 안팎의 도시 기반 가상 리그
- 장마, 여름 원정, 가을 야구, 군 복무/상무식 육성 구조를 연상시키는 시스템
- 응원 문화, 불펜 운영, 외국인 선수, 신인 드래프트, 2군 콜업을 가상화
- 국가대표 영광의 순간을 "Glory Scenario"라는 가상 역사 챌린지로 재해석

## Genre

- Realistic baseball action simulation
- Single-player and local practice first
- Online 1v1 later
- PC first, console-ready input architecture

## Core Pillars

1. **직접 플레이가 먼저다**
   - 타격, 투구, 수비, 주루 모두 조작 가능한 게임이어야 한다.
   - 감독/운영은 액션 야구를 강화하는 레이어다.

2. **공이 믿을 만해야 한다**
   - 구속, 회전축, 회전수, 릴리스, 공기 저항, Magnus 효과, 바운드, 배트 충돌을 커스텀 물리로 계산한다.
   - 결과는 RNG가 아니라 입력과 선수 능력, 물리 상태의 합으로 설명돼야 한다.

3. **선수 동작이 야구다워야 한다**
   - 스윙 궤도, 투구폼, 포구 자세, 송구 모션이 게임 결과와 연결돼야 한다.
   - 모션은 장식이 아니라 판정의 일부다.

4. **한국 야구의 공기를 담는다**
   - 실제 IP를 쓰지 않아도, 야간 경기 조명, 응원 리듬, 더그아웃 감정, 가을 경기 압박감이 느껴져야 한다.

5. **온라인은 결과보다 입력을 신뢰해야 한다**
   - 공과 배트 충돌은 서버 권위 시뮬레이션으로 판정한다.
   - 클라이언트는 예측 렌더링을 하되, 최종 결과는 서버의 고정 틱 시뮬레이션을 따른다.

## Fictional League

### Working League Name

KBL을 직접 쓰지 않는다. 후보:

- 대한 베이스볼 리그 / DBL
- 코리아 프로 베이스볼 / KPB
- 퍼스트 나인 리그 / FNL

### Initial Teams

- 서울 브릿지스
  - 팀 성향: 선발 야구, 정교한 내야 수비
  - 구장 이미지: 강변 야간 조명, 넓은 외야

- 부산 하버스
  - 팀 성향: 장타, 강한 불펜, 공격적 주루
  - 구장 이미지: 바람, 원정 응원, 짧은 우측 담장

초기에는 두 팀으로 수직 절편을 만든 뒤 10-12개 팀으로 확장한다.

## Core Gameplay

### Pitching

1. 구종 선택: 직구, 투심, 커터, 슬라이더, 커브, 체인지업, 스플리터.
2. 목표 존 선택: 9분할 존 + 유인구 영역.
3. 투구 강도 선택: 전력, 밸런스, 제구 우선.
4. 릴리스 입력: 아날로그 방향/타이밍 기반.
5. 결과: 릴리스 오차, 회전축, 회전수, 구속, 제구, 포수 포구 난이도.

### Batting

1. 타격 접근: 컨택, 밸런스, 파워, 커트.
2. 노림: 존, 구종, 방향성.
3. 스윙 입력: 타이밍 + 아날로그 배트 경로.
4. 충돌 판정: 배트 스윙 궤도와 공 궤적의 실제 교차.
5. 결과: 헛스윙, 파울, 먹힌 타구, 라인드라이브, 땅볼, 뜬공, 홈런.

### Fielding

1. 타구 판단과 첫 발 반응.
2. 낙구 지점 예측 보조.
3. 포구 타이밍, 다이빙, 점프, 펜스 플레이.
4. 송구 방향과 강도 입력.
5. 중계 플레이, 컷오프, 홈 송구, 병살 전환.

### Baserunning

1. 리드 폭과 스타트.
2. 태그업, 추가 진루, 귀루.
3. 슬라이딩 방향 선택.
4. 주자별 판단 보조 AI.

## Modes

### Exhibition

가상 구단으로 9이닝 경기를 플레이한다.

### Season

가상 한국 리그를 144경기 또는 축약 시즌으로 플레이한다.

### Career

투수 또는 타자 한 명을 키운다. 2군, 콜업, 포지션 변경, 부상, FA를 다룬다.

### Franchise

단장/감독 모드. 드래프트, 외국인 선수, 트레이드, 불펜 운영, 팬 여론을 다룬다.

### Glory Scenario

실제 국가대표 명장면을 직접 쓰지 않고, 구조만 가상화한 챌린지 모드.

예:
- **9-0**: 9전 전승 금메달을 연상시키는 무패 토너먼트
- 9회초 4득점 역전극을 연상시키는 원정 준결승
- 준우승으로 끝난 결승 재도전 시나리오
- 한일전 압박감만 가져온 라이벌전

## Vertical Slice

### Target

UE5에서 실제 3D 경기 장면으로 "이 방향이면 된다"를 증명한다. 기본 도형, 회색 Mannequin, 찰흙 블록아웃은 vertical slice가 아니다.

### Scope

- 한 구장
- 투수 1명, 타자 3명, 포수 1명, 내야수 4명, 외야수 3명
- 타격/투구/수비/주루 기본 조작
- 9회말 2아웃 1점 차 상황
- 실제 3D 공 궤적과 타구 결과
- 리플레이 카메라
- 기본 관중 반응과 타구음
- 리얼 선수 skeletal mesh
- 야구 특화 투구/타격/포수 모션캡처
- 배트, 공, 미트 소켓과 애니메이션 노티파이

세부 품질 기준은 [Realistic Vertical Slice Plan](REALISTIC_VERTICAL_SLICE.md)을 따른다.

### Success Criteria

- 스트라이크와 볼이 눈으로 납득된다.
- 헛스윙과 정타가 입력 타이밍/배트 경로로 설명된다.
- 타구가 억지 애니메이션이 아니라 물리 결과처럼 보인다.
- 수비수가 공을 따라가는 움직임이 게임처럼 빠르되 야구답다.
- 한 타석이 끝난 뒤 바로 다시 플레이하고 싶어야 한다.
- 첫 화면에서 야구장, 선수, 배트, 미트, 투구 자세가 리얼 야구 게임 방향으로 읽힌다.
